# Employee Management API

Spring Boot REST API project for managing employee records.

## Features
- Add Employee
- Get All Employees
- Get Employee By ID
- Update Employee
- Delete Employee
- Validation Handling
- Exception Handling

## Tech Stack
- Java 25
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Tools Used
- Postman
- GitHub
- Eclipse

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /employees | Add Employee |
| GET | /employees | Get All Employees |
| GET | /employees/{id} | Get Employee By ID |
| PUT | /employees/{id} | Update Employee |
| DELETE | /employees/{id} | Delete Employee |
